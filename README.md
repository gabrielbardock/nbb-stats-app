# 🏀 NBB Stats App

Aplicação web para visualização de estatísticas do **Novo Basquete Brasil (NBB)**, consumindo dados em tempo real a partir de uma API própria.

O projeto permite filtrar estatísticas por temporada, fase, categoria e tipo, além de buscar jogadores ou equipes de forma dinâmica.

---

## 🚀 Tecnologias Utilizadas

- **React** + **TypeScript**
- **Vite**
- **CSS puro**
- **API REST** (FastAPI – Python)

---

## 📊 Funcionalidades

- 🔍 Busca por **jogador** ou **equipe**
- 🎯 Filtros dinâmicos:
  - Temporada
  - Fase (Regular / Playoffs)
  - Categoria estatística
  - Tipo (média, total, etc.)
- 📄 Tabela responsiva de estatísticas
- ⚡ Carregamento otimizado
- 🌐 Consumo de API externa
